Built for Arabic/English slide decks

Fast translation with correct RTL/LTR layout behavior.

This add-on helps you translate slides between Arabic and English, while keeping the design readable by applying layout flipping rules (RTL ↔ LTR) and preserving structure.

How it works v1.0
Note: This website is informational. The add-on runs inside Google Workspace.

Translate (AR ↔ EN)

Detects source language and translates text elements while preserving intent and readability.

Flip Layout (RTL ↔ LTR)

Repositions shapes and text boxes to mirror the slide layout for the target reading direction.

Safe by default

Designed to minimize data retention. See the Privacy page for details on handling and storage.

Operator-friendly

Clear status messages and predictable behavior to reduce rework when translating decks at scale.

Preferences (for demonstration)

Auto-detect language Helps avoid translating to the same language.
Flip layout direction Mirrors positions between RTL and LTR.
